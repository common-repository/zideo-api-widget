Easy to use plugin to integrate Zideo Video's in your blog.

== Description ==

This plugin enables the use of Zideo embedded video's in your sidebar as a widget. 

It also enables an include tag to be used in your pages to embed the video's there aswell.
This is an PHP script tag, so the include on pages function has a plugin dependancy called "Include It".

The Settings page of the plugin has more explanation about how to use and configure the plugin.


== Installation ==

1. Upload all the files and folders to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the Settings menu to configure the plugin
